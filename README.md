# Clinicile Pink - Dr. Cătălina Toader

Website pentru Clinicile Pink - Stomatologie Copii, dezvoltat de sky.ro.

## Descriere

Site-ul prezintă serviciile de stomatologie pediatrică oferite de Dr. Cătălina Toader, incluzând:
- Profilaxie
- Tratament carii
- Ortodonție
- Informații despre locații și contact

## Caracteristici

- Design responsive pentru mobil și desktop
- Bară de promoție fixă la partea de sus
- Animații smooth scroll
- Carusel de servicii pentru mobil
- Carduri de locație cu hărți interactive
- Formular de contact
- Modal cu lista de prețuri

## Tehnologii

- HTML5
- Tailwind CSS (via CDN)
- Font Awesome (via CDN)
- JavaScript vanilla

## Deploy pe Netlify

1. Conectează acest repository la Netlify
2. Setează directorul de publicare la `.` (root)
3. Deploy automat la fiecare push pe branch-ul principal
